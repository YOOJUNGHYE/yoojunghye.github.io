PLAN PURPOSE

매일 8시간 5개월간 동안 만들었던 저의 코딩 작업물입니다. 주로 기존 사이트를 벤치마킹하여 그대로
따라하는 방식으로 코딩을 습득했습니다. 기존 사이트와 최대한 비슷하게 만들기 위해 연구를 하고
기능들을 살펴보면서 만들었던 홈페이지입니다.

Web Site 01 _ 더 프라자 호텔
더 프라자 호텔을 홈페이지를 참고하여, 기존에 있던 홈페이지 구조에
맞춰 똑같이 구현하도록 노력한 홈페이지 입니다. 배경이 자연스럽게
여러 이미지로 바뀌도록 했으며, 제이쿼리를 이용해서 마우스가
hover 됐었을때의 동작도 구현했습니다.

Web Site 02 _ 이랜드 서비스
이랜드 서비스 홈페이지를 참고하여, 기존에 있던 홈페이지 구조에
맞춰 똑같이 구현하도록 노력한 홈페이지 입니다. 메인 제목이
자연스럽게 움직이도록 구현했으며, 홈페이지를 아래로 scroll
할때마다 한 페이지씩 보이도록 만들었습니다. 아래로 내릴 수록
각기 다른 움직임이 있으며 마지막 footer 부분은 슬라이더가
넘어가도록 만들었습니다.

Web Site 03 _ 한국애플서비스센터
회사에서 애플 제품 전용 전문서비스센터 홈페이지를 만들었습니다.
홈페이지를 기획할때 ‘애플 홈페이지가 가장 신뢰를 주겠다.’ 생각이
들어서 가장 애플 홈페이지 비슷하게 만들었습니다. 메뉴 버튼도
벤치마킹하여 호버가 되면 커지게 했고, 전반적으로 블랙의 느낌으로
만들었습니다.

Web Site 04 _ 애플서비스
위의 홈페이지와 동일한 기획하에 만들어진 홈페이지며,
화이트 톤의 느낌으로 만든 홈페이지 입니다.

Web Site 05 _ 에어부산
에어 부산의 모바일 버전을 벤치마킹하여 제이쿼리 모바일로
코딩하여 만든 홈페이지 입니다.

Web Site 06 _ 대한항공
대한 항공의 모바일 버전을 벤치마킹하여 제이쿼리 모바일로
코딩하여 만든 홈페이지 입니다.
